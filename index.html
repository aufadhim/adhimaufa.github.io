Line Follower Analog

Pcb Design

This board is made using autodesk eagle software. This software can be used free of charge using education license. We make a schematic first by choosing what components to use. After that, do the routing process. The picture above is part of the main board, this is where we put the microcontroller as the brain of the line follower robot. Then the picture below is part of the board for the line follower sensor. The sensor used is a diode sensor, there are 14 sensors overthere.

  • Date: Desember 2017
  • Client: Student
  • Category: Printed Circuit Board Design

PCB Design
